## Reporting a Vulnerability

We take the security of our agricultural data and user privacy seriously. If you discover a security vulnerability within this project, please follow these steps:

Do not open a GitHub Issue. Disclosing vulnerabilities in public issues can expose users to unnecessary risk.

Contact the Maintainer. Please send a detailed report to [jdmyersuw76@gmail.com].

Include Details. To help us resolve the issue quickly, please include:

A description of the vulnerability.

Steps to reproduce the issue (PoC).

Potential impact if exploited.

Our Commitment
We will acknowledge receipt of your report within 48 hours.

We will provide an estimated timeline for a fix.

Once the vulnerability is resolved, we will notify you and (with your permission) credit you for the discovery.

Preferred Communication
For sensitive disclosures, you may contact the project lead via the Cyber Sanctum professional portal.
